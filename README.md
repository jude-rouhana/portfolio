# Portfolio Website

A personal portfolio website for Jude Rouhana, built by Jude Rouhana. This site showcases projects, work, music, and provides a way to get in touch.

## Tools & Technologies

- **React 19** - UI framework
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Router DOM** - Client-side routing
- **Three.js** & **React Three Fiber** - 3D graphics and interactive experiences
- **GSAP** - Advanced animation library
- **React Intersection Observer** - Scroll-based animations
- **EmailJS** - Contact form functionality
- **PostCSS** & **Autoprefixer** - CSS processing

## Content

The portfolio features several sections:

- **Home** - Landing page with interactive elements and animations, use canvas mode to create your own pixel art
- **About** - Personal background, skills, and experience
- **Projects** - Showcase of coding projects and work, including detailed descriptions and media
- **Music** - Audio content and track listings
- **Games** - Interactive game projects and demos
- **Contact** - Contact form and ways to connect
- **Experimental** - Experimental features and creative explorations

The site includes smooth animations, dark/light mode support, responsive design, and interactive elements throughout.
